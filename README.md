## Design

- **Format**: 540 × 675 px per slide (4:5 ratio, mobile-optimized)
- **Font**: Inter (Google Fonts)
- **Color palette**:
  - Ocean Blue `#0B4F6C`
  - Jeju Teal `#00B4CC`
  - Accent Orange `#FF6B35`
  - Dark `#0D1B2A`
- **Images**: Official facility photos from [jeju-sp.com](https://www.jeju-sp.com/english/innovation/facility/starter.htm) (used with permission)

---

## Usage

Open `jeju-innovation-center.html` directly in any browser. No build step required.

To export individual slides as PNG, open in Chrome → right-click each `.slide` element → Screenshot via DevTools.
